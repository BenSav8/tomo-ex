# tomo-ex

this is a custom font for trollbox and rmtrollbox, it fixes inconsistencies and adds new characters

# how to use

- if you use tampermonkey (RECOMMENDED):
  - add the corresponding _-tampermonkey.js file as an userscript (rmtb-tampermonkey.js for rmtrollbox, tb-tampermonkey.js for w93 trollbox)
- if you use [TBJB](https://github.com/BenSav8/tbjb-tampermonkey):
  - download the tomo-ex.js file from this repository
  - do /store then add it as an addon
- if you're on normal trollbox
  - open devtools (inspect element)
  - go on console
  - paste the contents of tomo-ex.js
